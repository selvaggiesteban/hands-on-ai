SEO in 2026 is driven by SGE (Search Generative Experience) and E-E-A-T (Experience, Expertise, Authoritativeness, Trustworthiness).
The content_strategist acts as the primary authority on implementation and quality standards within this domain.

All work produced must adhere to the following non-negotiable standards:
- [MANDATE]: Semantic HTML5 is the foundation of technical SEO; no exceptions.
- [MANDATE]: Content must satisfy both user intent and Large Language Model (LLM) indexing requirements.
- [MANDATE]: Structured Data (Schema.org) must be implemented using JSON-LD for every entity.
- [MANDATE]: Entity-based SEO over keyword density: focus on Knowledge Graph integration.
- [MANDATE]: Automated link-building must strictly adhere to white-hat 'Authority-First' protocols.
