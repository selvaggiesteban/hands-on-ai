# Accessibility

- Is the application navigable via keyboard? [YES/NO]
- Are 'alt' attributes present on all images? [YES/NO]
- Is color contrast sufficient (WCAG AA)? [YES/NO]
- Are semantic HTML tags used? [YES/NO]
- Are ARIA labels used where semantic HTML is insufficient? [YES/NO]
- Is the application tested with a screen reader? [YES/NO]
