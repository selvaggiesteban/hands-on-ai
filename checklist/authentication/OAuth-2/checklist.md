# OAuth 2.0

- Is the 'state' parameter used to prevent CSRF? [YES/NO]
- Is PKCE (Proof Key for Code Exchange) used for public clients? [YES/NO]
- Are scopes limited to the minimum required? [YES/NO]
- Is the redirect URI validated strictly? [YES/NO]
- Are access tokens stored securely (not in local storage if possible)? [YES/NO]
- Is the Authorization Code flow preferred over Implicit flow? [YES/NO]
