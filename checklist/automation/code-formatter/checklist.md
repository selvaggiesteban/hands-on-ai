# Code Formatter

- Is a configuration file (e.g., .prettierrc, .clang-format) present in the root directory? [YES/NO]
- Does the formatter run automatically on file save in the IDE? [YES/NO]
- Is there a pre-commit hook to ensure all staged files are formatted? [YES/NO]
- Is the line width standardized (e.g., 80 or 120 characters) across the project? [YES/NO]
- Are indentation rules (spaces vs. tabs) consistent and enforced? [YES/NO]
- Does the CI pipeline fail if code is not properly formatted? [YES/NO]
