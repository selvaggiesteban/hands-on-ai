# General Backend Standards

- Is the API RESTful or GraphQL compliant with established standards? [YES/NO]
- Are database queries optimized to avoid N+1 problems? [YES/NO]
- Is error handling centralized and consistent across the application? [YES/NO]
- Are logs structured (e.g., JSON) and contain sufficient context (Correlation IDs)? [YES/NO]
- Is the application stateless to allow for horizontal scaling? [YES/NO]
- Are background jobs used for long-running tasks? [YES/NO]
