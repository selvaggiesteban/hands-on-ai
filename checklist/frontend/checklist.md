# General Frontend Standards

- Is the UI responsive and accessible (WCAG 2.1 AA compliant)? [YES/NO]
- Are assets (images, scripts, styles) optimized and compressed? [YES/NO]
- Is state management handled predictably (e.g., Redux, Context, Vuex)? [YES/NO]
- Are component render cycles optimized to prevent performance bottlenecks? [YES/NO]
- Is input sanitization performed to prevent XSS attacks? [YES/NO]
- Are modern browser features used with appropriate polyfills for legacy support? [YES/NO]
