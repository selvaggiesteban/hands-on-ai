# Git Branches

- Is a branching strategy (GitFlow, Trunk-Based) followed? [YES/NO]
- Are branch names descriptive (feature/..., fix/...)? [YES/NO]
- Are stale branches deleted regularly? [YES/NO]
- Is the main branch protected? [YES/NO]
- Are feature branches kept short-lived? [YES/NO]
- Is rebasing preferred over merging for clean history (optional)? [YES/NO]
