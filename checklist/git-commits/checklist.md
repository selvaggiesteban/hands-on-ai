# Git Commits

- Is the commit message clear and descriptive? [YES/NO]
- Does the commit follow the Conventional Commits specification? [YES/NO]
- Is the commit atomic (one logical change per commit)? [YES/NO]
- Are all temporary debugging statements (e.g., console.log) removed? [YES/NO]
- Is the commit associated with a task or issue ID? [YES/NO]
- Do all tests pass locally before committing? [YES/NO]
