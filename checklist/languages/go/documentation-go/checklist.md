# Documentation

- Do exported types, functions, and methods have doc comments? [YES/NO]
- Do doc comments start with the name of the exported identifier? [YES/NO]
- Are package comments present in at least one file per package? [YES/NO]
- Are examples provided for complex functions using 'Example' test functions? [YES/NO]
- Is 'godoc' formatting rules followed? [YES/NO]
- Are deprecated identifiers marked with '// Deprecated:'? [YES/NO]
