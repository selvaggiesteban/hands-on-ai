# Interfaces

- Are interfaces defined where they are used (consumer-side)? [YES/NO]
- Are interfaces kept small (single method preferred)? [YES/NO]
- Is the empty interface 'interface{}' or 'any' used sparingly? [YES/NO]
- Is interface implementation implicit (duck typing) understood? [YES/NO]
- Are interface values checked for nil correctly (both type and value)? [YES/NO]
- Is embedding used to compose interfaces? [YES/NO]
