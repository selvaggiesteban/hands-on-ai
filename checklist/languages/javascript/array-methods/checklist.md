# Array Methods

- Are 'map', 'filter', and 'reduce' used instead of manual loops? [YES/NO]
- Is 'find' used for searching single elements? [YES/NO]
- Is 'some' or 'every' used for boolean checks? [YES/NO]
- Are mutating methods (push, pop, splice) avoided when immutability is required? [YES/NO]
- Is 'includes' used instead of 'indexOf' for presence checks? [YES/NO]
- Are 'flatMap' used for mapping and flattening in one pass? [YES/NO]
