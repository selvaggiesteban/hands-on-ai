# Arrow Functions

- Are arrow functions used for callbacks (to preserve 'this')? [YES/NO]
- Is implicit return used for one-liners? [YES/NO]
- Are parentheses used around arguments only when necessary (or always if enforcing consistency)? [YES/NO]
- Are arrow functions avoided for object methods (where 'this' is needed)? [YES/NO]
- Are arrow functions avoided for defining prototypes? [YES/NO]
- Is readability maintained when nesting arrow functions? [YES/NO]
