# Async/Await

- Is await always wrapped in a try/catch block or handled with .catch()? [YES/NO]
- Are multiple independent async operations executed in parallel with Promise.all()? [YES/NO]
- Is await avoided inside loops (preferring Promise.all or for...of for sequentiality)? [YES/NO]
- Are async functions returning a Promise explicitly typed if using TypeScript? [YES/NO]
- Is there a mechanism to handle potential 'unhandled promise rejections'? [YES/NO]
- Are timeouts implemented for async operations that may hang indefinitely? [YES/NO]
