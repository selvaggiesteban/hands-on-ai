# JavaScript Standards

- Is the code written using modern ECMAScript standards (ES6+)? [YES/NO]
- Are 'let' and 'const' used instead of 'var'? [YES/NO]
- Is async/await used for asynchronous operations to avoid callback hell? [YES/NO]
- Are strict equality operators (=== and !==) used consistently? [YES/NO]
- Is the code free of console.log statements in production? [YES/NO]
- Are dependencies managed using npm or yarn with a lockfile? [YES/NO]
