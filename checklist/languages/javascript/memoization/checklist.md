# Memoization

- Is memoization used for expensive function calls? [YES/NO]
- Are cache keys generated correctly for multiple arguments? [YES/NO]
- Is memory usage monitored (cache invalidation/eviction)? [YES/NO]
- Are libraries like 'reselect' (Redux) or 'useMemo' (React) used? [YES/NO]
- Is memoization avoided for cheap or side-effect-heavy functions? [YES/NO]
- Is the cache pure (same input = same output)? [YES/NO]
