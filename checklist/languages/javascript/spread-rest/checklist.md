# Spread and Rest

- Is spread syntax used for copying arrays and objects (shallow copy)? [YES/NO]
- Is rest syntax used for variable number of function arguments? [YES/NO]
- Is spread used for merging objects? [YES/NO]
- Are the limitations of shallow copying understood? [YES/NO]
- Is spread used in place of 'apply' for passing arrays as arguments? [YES/NO]
- Is rest parameter always the last parameter in the function definition? [YES/NO]
