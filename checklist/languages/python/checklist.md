# Python Standards

- Does the code adhere to PEP 8 style guidelines? [YES/NO]
- Are type hints used to improve code readability and maintainability? [YES/NO]
- Is a virtual environment (e.g., venv, poetry) used for dependency management? [YES/NO]
- Are docstrings provided for all public modules, functions, and classes? [YES/NO]
- Is error handling implemented using specific exception types? [YES/NO]
- Does the project use a linter (e.g., Ruff, Flake8) and a formatter (e.g., Black)? [YES/NO]
