# Docstrings

- Are docstrings written in valid ReStructuredText (or Google/NumPy style)? [YES/NO]
- Do all public modules, classes, and functions have docstrings? [YES/NO]
- Does the docstring describe arguments, return values, and exceptions? [YES/NO]
- Are type information included in docstrings if not using type hints? [YES/NO]
- Is 'doctest' used to verify code examples in docstrings? [YES/NO]
- Are docstrings kept up-to-date with code changes? [YES/NO]
