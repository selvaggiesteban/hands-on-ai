# Package Management

- Is a dependency manager (pip, poetry, pipenv) used? [YES/NO]
- Is 'requirements.txt' or 'pyproject.toml' kept up-to-date? [YES/NO]
- Are virtual environments used to isolate dependencies? [YES/NO]
- Are packages structured with '__init__.py' files? [YES/NO]
- Is a proper build backend configured (e.g., setuptools, poetry-core)? [YES/NO]
- Are unused dependencies removed regularly? [YES/NO]
