# Type Hints

- Are type hints used for function arguments and return values? [YES/NO]
- Is the 'typing' module (or standard collection types in 3.9+) used? [YES/NO]
- Is 'Optional', 'Union', 'List', 'Dict' used correctly? [YES/NO]
- Are type aliases used for complex types? [YES/NO]
- Is a static type checker (mypy, pyright) run in CI? [YES/NO]
- Are stubs (.pyi) used for untyped libraries? [YES/NO]
