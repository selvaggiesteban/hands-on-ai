# Borrowing

- Are references (&) used instead of cloning where possible? [YES/NO]
- Is mutable borrowing (&mut) restricted to one scope at a time? [YES/NO]
- Are lifetimes explicit only when the compiler cannot infer them? [YES/NO]
- Is 'RefCell' used for interior mutability only when necessary? [YES/NO]
- Are slice references used for contiguous memory access? [YES/NO]
- Is the borrow checker satisfied without 'unsafe' code? [YES/NO]
