# Rust Standards

- Does the code compile without warnings under 'cargo check'? [YES/NO]
- Are ownership and borrowing rules respected to avoid unnecessary clones? [YES/NO]
- Is error handling performed using Result and Option types instead of panics? [YES/NO]
- Is 'clippy' used for linting and enforcing idiomatic Rust patterns? [YES/NO]
- Are public APIs documented using doc-comments (///)? [YES/NO]
- Is 'cargo fmt' used to maintain consistent code formatting? [YES/NO]
