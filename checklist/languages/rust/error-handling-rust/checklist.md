# Error Handling

- Is 'Result<T, E>' used for recoverable errors? [YES/NO]
- Is 'Option<T>' used for absence of value? [YES/NO]
- Is the '?' operator used for error propagation? [YES/NO]
- Are custom error types defined using 'thiserror' or 'anyhow'? [YES/NO]
- Are panics avoided in production code? [YES/NO]
- Is 'unwrap()' avoided unless safety is guaranteed? [YES/NO]
