# Smart Pointers

- Is 'Box<T>' used for heap allocation? [YES/NO]
- Is 'Rc<T>' used for reference counting within a thread? [YES/NO]
- Is 'Arc<T>' used for atomic reference counting across threads? [YES/NO]
- Is 'RefCell<T>' used for interior mutability? [YES/NO]
- Is 'Cow<T>' (Copy on Write) used for optimization? [YES/NO]
- Are smart pointers dereferenced automatically? [YES/NO]
