# Testing

- Are unit tests placed in the same file as the code (cfg(test))? [YES/NO]
- Are integration tests placed in the 'tests' directory? [YES/NO]
- Is 'assert_eq!' and 'assert_ne!' used? [YES/NO]
- Are 'should_panic' attributes used for expected failures? [YES/NO]
- Are doctests used to verify documentation examples? [YES/NO]
- Is 'cargo test' used to run all tests? [YES/NO]
