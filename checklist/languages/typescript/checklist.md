# TypeScript Standards

- Is 'strict' mode enabled in tsconfig.json? [YES/NO]
- Are interfaces or types used for all data structures and API responses? [YES/NO]
- Is the 'any' type avoided in favor of more specific types or 'unknown'? [YES/NO]
- Are enums or literal types used for fixed sets of values? [YES/NO]
- Is the project configured to fail the build on TypeScript errors? [YES/NO]
- Are decorators used consistently if required by the framework (e.g., NestJS, Angular)? [YES/NO]
