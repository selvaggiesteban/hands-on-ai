# Modules

- Is 'export type' used to export types explicitly? [YES/NO]
- Are 'import type' used to import types (erased at runtime)? [YES/NO]
- Are ES modules syntax used consistently? [YES/NO]
- Is 'moduleResolution' set to 'node' or 'bundler'? [YES/NO]
- Are path aliases (tsconfig 'paths') used for cleaner imports? [YES/NO]
- Is 'allowSyntheticDefaultImports' configured if needed? [YES/NO]
