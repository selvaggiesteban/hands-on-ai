# Strict Mode

- Is 'strict: true' enabled in tsconfig.json? [YES/NO]
- Is 'noImplicitAny' resolved everywhere? [YES/NO]
- Is 'strictNullChecks' handled with unions (Type | null)? [YES/NO]
- Is 'strictPropertyInitialization' satisfied in classes? [YES/NO]
- Is 'noImplicitThis' handled? [YES/NO]
- Is 'useUnknownInCatchVariables' enabled? [YES/NO]
