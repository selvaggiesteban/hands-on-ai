# TSConfig

- Is 'target' set to an appropriate ECMAScript version? [YES/NO]
- Is 'lib' configured to include necessary environment types (DOM, ESNext)? [YES/NO]
- Is 'outDir' and 'rootDir' configured for build output? [YES/NO]
- Is 'incremental' builds enabled for faster compile times? [YES/NO]
- Are 'types' and 'typeRoots' configured correctly? [YES/NO]
- Is 'skipLibCheck' enabled to speed up compilation? [YES/NO]
