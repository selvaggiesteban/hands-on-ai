# Type Inference

- Is type inference relied upon for simple variable initializations? [YES/NO]
- Are return types inferred where obvious, but explicit for public APIs? [YES/NO]
- Is 'const' assertion used to infer literal types? [YES/NO]
- Is 'satisfies' operator used to check type without widening (TS 4.9+)? [YES/NO]
- Are generic type parameters inferred from arguments? [YES/NO]
- Is context-sensitive typing leveraged in callbacks? [YES/NO]
