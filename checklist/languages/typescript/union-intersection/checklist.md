# Union and Intersection

- Are union types used for values that can be one of several types? [YES/NO]
- Are intersection types used to combine multiple types? [YES/NO]
- Is narrowing used to handle union members? [YES/NO]
- Are discriminated unions used for state management? [YES/NO]
- Is 'never' type understood as the empty union? [YES/NO]
- Are parentheses used to clarify precedence in complex combinations? [YES/NO]
