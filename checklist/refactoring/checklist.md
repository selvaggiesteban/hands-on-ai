# Refactoring

- Is refactoring done separately from feature work? [YES/NO]
- Are tests passing before and after refactoring? [YES/NO]
- Is the goal (readability, performance) clear? [YES/NO]
- Is code duplication reduced (DRY)? [YES/NO]
- Is complexity reduced (KISS)? [YES/NO]
- Is the 'Boy Scout Rule' applied (leave code better than found)? [YES/NO]
