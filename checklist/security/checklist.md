# Security Audit

- Are all user inputs sanitized to prevent injection attacks? [YES/NO]
- Is sensitive data (secrets, keys) stored outside of the codebase? [YES/NO]
- Are all third-party dependencies up-to-date and free of known vulnerabilities? [YES/NO]
- Is authentication required for all protected resources? [YES/NO]
- Are session tokens handled securely (HttpOnly, Secure, etc.)? [YES/NO]
- Are there logs for security-relevant events (failed logins, etc.)? [YES/NO]
