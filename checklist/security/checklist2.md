# General Security

- Are all secrets and API keys excluded from version control (e.g., in .env files)? [YES/NO]
- Is HTTPS enforced for all external communications? [YES/NO]
- Are dependencies scanned for known vulnerabilities (e.g., using Snyk or Dependabot)? [YES/NO]
- Is input validation performed on all API endpoints to prevent injection attacks? [YES/NO]
- Are proper authentication and authorization checks applied to all protected routes? [YES/NO]
- Is sensitive data (passwords, PII) encrypted at rest and in transit? [YES/NO]
