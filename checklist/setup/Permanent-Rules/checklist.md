# Permanent Rules

- Are non-negotiable rules (e.g., security, legal) clearly identified? [YES/NO]
- Is compliance with permanent rules audited regularly? [YES/NO]
- Are exceptions to permanent rules handled via a formal waiver process? [YES/NO]
- Is there a mechanism to update permanent rules when necessary? [YES/NO]
- Are permanent rules visible to all team members? [YES/NO]
- Is automated enforcement (linting, policy-as-code) used where possible? [YES/NO]
