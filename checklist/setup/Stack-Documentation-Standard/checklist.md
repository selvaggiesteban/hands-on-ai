# Stack Documentation Standard

- Is the technology stack fully documented? [YES/NO]
- Are decision records (ADRs) kept for technology choices? [YES/NO]
- Is the documentation kept close to the code? [YES/NO]
- Are diagrams (C4 model, sequence) used? [YES/NO]
- Is the documentation reviewed as part of the PR process? [YES/NO]
- Are deprecated technologies flagged? [YES/NO]
