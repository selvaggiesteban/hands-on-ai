# Apache

- Is 'mod_security' enabled for WAF protection? [YES/NO]
- Is 'mod_evasive' configured for DoS protection? [YES/NO]
- Are directory listings disabled ('Options -Indexes')? [YES/NO]
- Is '.htaccess' usage minimized for performance (move to vhosts)? [YES/NO]
- Are server tokens hidden ('ServerTokens Prod')? [YES/NO]
- Is HTTP/2 enabled? [YES/NO]
