# Nginx

- Is Gzip or Brotli compression enabled? [YES/NO]
- Are 'server_tokens' set to 'off'? [YES/NO]
- Is a strong SSL configuration (Mozilla intermediate/modern) used? [YES/NO]
- Are buffer overflow protections configured (client_body_buffer_size)? [YES/NO]
- Is upstream load balancing configured correctly? [YES/NO]
- Are static assets cached with long expiry times? [YES/NO]
