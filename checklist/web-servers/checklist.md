# Web Server Configuration

- Is the web server configured to use modern TLS versions (1.2 or 1.3)? [YES/NO]
- Are security headers (HSTS, CSP, X-Frame-Options) enabled? [YES/NO]
- Is request rate limiting implemented to mitigate DoS attacks? [YES/NO]
- Are access logs and error logs enabled and rotated? [YES/NO]
- Is the server configured to hide version information in headers? [YES/NO]
- Are static assets served with appropriate caching headers? [YES/NO]
