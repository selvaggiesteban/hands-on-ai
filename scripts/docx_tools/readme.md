# Word Document Tools (DOCX)

Colección de scripts para automatizar el procesamiento de archivos Microsoft Word.

## Funcionalidades
*   **ooxml:** Validación de estructuras XML frente a esquemas oficiales.
*   **scripts:** Lógica para manipular el contenido del documento.
*   **templates:** Plantillas base para la generación de documentos.
