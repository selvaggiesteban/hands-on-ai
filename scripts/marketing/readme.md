# Herramientas de Marketing

Colección de scripts y utilidades para potenciar operaciones de marketing digital.

## Herramientas Incluidas

### SEO y Contenido
*   **seo_content_generator:** Generación automática de contenido optimizado.
*   **editorial_calendar:** Gestión y planificación de calendario editorial.
*   **chatgpt_to_pdf:** Conversión de conversaciones de IA a documentos portables.

### Scraping y Datos
*   **scraper_woocommerce:** Extracción de datos de tiendas WooCommerce.
*   **scraper_googlebusinessprofile:** Obtención de información de perfiles de negocio.
*   **scraper_webtopdf:** Captura de sitios web a PDF.
*   **googlebusinessprofile_leads:** Generación de leads desde Google Maps.

### Utilidades y Migración
*   **contabilidad_dashboard:** Panel de control financiero/contable.
*   **landing_page_generator:** (Antes `google.selvaggiesteban.dev`) Generador de landing pages.
*   **migration_argentiniangooglebusinessprofiles:** Scripts específicos de migración.
*   **csv_to_excel:** Conversor de formatos de datos.
*   **web_mockup:** Generador de mockups visuales.
