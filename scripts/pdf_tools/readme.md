# Herramientas de Procesamiento de Documentos

Scripts para automatizar la creación, validación y modificación de documentos de oficina.

## Módulos

*   **pdf_tools:** Scripts para validación de formularios, extracción de campos y conversión de imágenes en PDFs.
*   **docx_tools:** Utilidades para manipular documentos Word y estructura OOXML subyacente.
*   **pptx_tools:** Herramientas para gestión de presentaciones PowerPoint, reordenamiento y validación OOXML.
