# PowerPoint Presentation Tools (PPTX)

Scripts diseñados para automatizar tareas complejas en presentaciones de PowerPoint.

## Funcionalidades
*   **inventory.py:** Genera un inventario detallado del contenido de las diapositivas.
*   **rearrange.py:** Permite el reordenamiento programático de slides.
*   **replace.py:** Automatiza el reemplazo de texto y activos en presentaciones.
*   **ooxml:** Validación técnica de la estructura de la presentación.
