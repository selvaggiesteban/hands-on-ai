# Skill Creator Tools

Herramientas para desarrolladores que permiten estandarizar la creación de nuevas capacidades para los agentes.

## Scripts Incluidos
*   **init_skill.py:** Scaffolding inicial de la estructura de una skill.
*   **package_skill.py:** Prepara una skill para su distribución o despliegue.
*   **quick_validate.py:** Verificación rápida de cumplimiento de formatos.
