# Slack GIF Creator

Herramienta para crear animaciones personalizadas y funcionales para la comunicación en Slack.

## Estructura
*   **core:** Motores de easing, composición de frames y lógica de construcción de GIFs.
*   **validators:** Asegura que los GIFs cumplan con las limitaciones de tamaño y formato de Slack.
