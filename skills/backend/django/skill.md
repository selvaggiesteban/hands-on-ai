# Django Development Skill

This skill provides expert capabilities in Django 4+ development, focusing on scalable architecture, ORM optimization, and secure API development.

## Capabilities

- **Django Architecture**: MVT pattern, App structure, Middleware pipeline
- **ORM Mastery**: Complex queries, Database optimization, Migrations
- **REST APIs**: Django REST Framework, Serialization, ViewSets
- **Security**: CSRF protection, XSS prevention, SQL injection defense

## Process

1.  **Context Assessment**: Analyze project requirements, database design, and API needs.
2.  **Architecture Planning**: Design scalable app structure and database schema.
3.  **Implementation**: Build robust models, views, and APIs following best practices.
4.  **Optimization**: Optimize queries, implement caching, and ensure performance.
5.  **Verification**: Execute comprehensive tests (pytest-django) and security checks.

## Best Practices

- Adhere to PEP 8 and Django coding style.
- Maintain test coverage > 90%.
- Use type hints for better code quality.
- Prioritize security and performance in every implementation.
