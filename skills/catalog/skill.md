# Catalog Skill

This skill serves as the central index and management point for all system capabilities, resources, and templates.

## Capabilities

- **Resource Indexing**: Cataloging available agents, skills, and tools.
- **Capability Mapping**: Mapping user requests to specific system capabilities.
- **Template Management**: Organizing and serving reusable project templates.
- **Documentation Serving**: Providing access to system documentation and guides.

## Process

1.  **Discovery**: Scan the system to identify available resources and agents.
2.  **Indexing**: Organize resources into logical categories (e.g., Backend, Frontend, DevOps).
3.  **Retrieval**: Efficiently locate and provide requested resources or information.
4.  **Maintenance**: Ensure the catalog reflects the current state of the system.

## Best Practices

- Maintain logical categorization consistent with file structure.
- Ensure all entries have clear descriptions and metadata.
- Regularly update the index to include new additions (agents/skills).
