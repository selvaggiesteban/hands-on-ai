# AWS Cloud Skill

This skill provides expert capabilities in AWS cloud architecture, focusing on the Well-Architected Framework, serverless patterns, and infrastructure as code.

## Capabilities

- **Compute & Serverless**: EC2, Lambda, ECS, Fargate
- **Storage & Database**: S3, RDS, DynamoDB, ElastiCache
- **Networking**: VPC design, Route53, CloudFront, API Gateway
- **Security**: IAM, KMS, Shield, WAF, Security Hub

## Process

1.  **Context Assessment**: Analyze business requirements and current infrastructure.
2.  **Architecture Design**: Design scalable, secure AWS solutions using Well-Architected principles.
3.  **Implementation**: Deploy resources using IaC (Terraform/CloudFormation/CDK).
4.  **Optimization**: Optimize for cost, performance, and security.
5.  **Verification**: Validate resilience, security posture, and compliance.

## Best Practices

- Apply "Security by Design" and least privilege principles.
- Use Infrastructure as Code for all deployments.
- Implement comprehensive monitoring and alerting.
- Regularly review cost optimization opportunities.
