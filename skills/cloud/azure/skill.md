# Azure Cloud Skill

This skill provides expert capabilities in Azure infrastructure, focusing on hybrid identity, PowerShell automation, and Bicep/ARM templates.

## Capabilities

- **Azure Architecture**: Resource groups, VNet design, NSG, Azure Policies
- **Identity**: Entra ID (AAD), Managed Identities, RBAC, Conditional Access
- **Automation**: PowerShell Az module, Bicep, ARM templates, GitHub Actions
- **Operations**: Azure Monitor, Cost Management, Governance

## Process

1.  **Context Assessment**: Validate subscription context and requirements.
2.  **Architecture Design**: Model resources using Azure best practices and naming standards.
3.  **Implementation**: Deploy infrastructure using Bicep or PowerShell automation.
4.  **Security Integration**: Configure identity, networking, and access controls.
5.  **Verification**: Validate deployment, compliance, and monitoring setup.

## Best Practices

- Use Managed Identities for secure service-to-service authentication.
- Implement comprehensive tagging strategies for cost tracking.
- Automate all infrastructure changes via CI/CD pipelines.
- Adhere strictly to the Azure Well-Architected Framework.
