# Next.js Development Skill

This skill provides expert capabilities in Next.js 14+ development, focusing on the App Router architecture, Server Components, and edge rendering.

## Capabilities

- **Core Architecture**: App Router, Layouts, Server/Client Components
- **Data & Rendering**: SSR, SSG, ISR, Streaming, Server Actions
- **Performance**: Core Web Vitals optimization, Image/Font optimization
- **SEO**: Metadata API, Sitemaps, Structured data

## Process

1.  **Context Assessment**: Define rendering strategy (Static vs Dynamic) and SEO requirements.
2.  **Architecture Design**: Structure routes, layouts, and data fetching patterns.
3.  **Implementation**: Build server-first components and implement Server Actions.
4.  **Optimization**: Optimize images, fonts, and scripts for maximum performance.
5.  **Verification**: Verify SEO scores and Core Web Vitals performance.

## Best Practices

- Default to Server Components for better performance and SEO.
- Use Server Actions for data mutations.
- Optimize images using the `next/image` component.
- Implement proper metadata for all routes.
