# Botanical Garden

A fresh and organic theme inspired by nature, using lush greens and earthy tones.

## Color Palette

- **Leaf Green**: `#4caf50` - Primary action color
- **Soil Brown**: `#795548` - Accents and borders
- **Cream**: `#f5f5f5` - Soft backgrounds
