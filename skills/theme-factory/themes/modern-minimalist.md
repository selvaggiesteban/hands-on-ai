# Modern Minimalist

Focus on content over chrome with a stark, clean aesthetic.

## Color Palette

- **Black**: `#000000` - Text and borders
- **White**: `#ffffff` - Negative space
- **Gray**: `#e0e0e0` - Subtle dividers
